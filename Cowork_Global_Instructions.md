# Global Instructions สำหรับ Claude Cowork

## บริบทผู้ใช้
ฉันเป็นนักวิจัยสังกัด School of ICT มหาวิทยาลัยพะเยา และ Hydro-Informatics Institute (HII)
งานหลักด้าน hydrology, agricultural water management, machine learning forecasting,
และ GIS/remote sensing เน้นพื้นที่ลุ่มน้ำภาคเหนือของไทย โดยเฉพาะตำบลแม่นาเรือ จังหวัดพะเยา

## งานที่กำลังทำอยู่ (Project: เว็บไซต์บริหารจัดการน้ำแม่นาเรือ)
กำลังสร้างเว็บไซต์ศูนย์ข้อมูลสนับสนุนการบริหารจัดการน้ำ ต.แม่นาเรือ อ.เมือง จ.พะเยา
รวมงานวิจัย 3 ด้าน: (1) พยากรณ์ความต้องการใช้น้ำเกษตร Zone A/B (2) พยากรณ์น้ำไหลลงอ่าง
(3) water monitoring จากสถานีโทรมาตร
รายละเอียดไอเดียฉบับสมบูรณ์อยู่ในไฟล์ "สรุปไอเดีย_เว็บไซต์บริหารจัดการน้ำ_แม่นาเรือ.md"
ให้อ่านไฟล์นี้ก่อนเริ่มงานใดๆ ที่เกี่ยวกับ project นี้

## ข้อเท็จจริงสำคัญที่ต้องใช้ถูกต้องเสมอ
- ชื่อคลองในระบบน้ำ ต.นครป่าหมาก คือ "แคววังทอง" (ไม่ใช่ "แควังทอง")
- โมเดลพยากรณ์ความต้องการใช้น้ำยังอยู่ระหว่างพัฒนา (validate ย้อนหลังเท่านั้น
  train 2020-2022, calibrate 2023, test 2024) ยังไม่ใช่ live forecast
  ห้ามนำเสนอเป็นระบบ real-time เต็มรูปแบบโดยไม่ระบุข้อจำกัดนี้
- ห้ามสร้างตัวเลขผลการทดลอง ([TBD] values) ขึ้นมาเองโดยไม่มีข้อมูลรองรับ

## สไตล์การทำงาน
- เขียนภาษาไทยเป็นหลัก แปลเป็นอังกฤษภายหลังเมื่อร้องขอ
- สื่อสารตรงประเด็น กระชับ ไม่ต้องอารัมภบทยาว
- งานเอกสารวิชาการใช้ร้อยแก้วแบบ continuous prose มากกว่า bullet point
  (ยกเว้นเอกสารสรุป/checklist ที่ bullet เหมาะกว่า)
- ก่อนแก้ไขหรือสร้างไฟล์สำคัญ ให้สรุปแผนสั้นๆ ก่อนลงมือ
- ถามก่อนถ้าจะลบไฟล์ หรือเขียนทับข้อมูลที่มีอยู่แล้ว

## การจัดหมวดข้อมูล (สำคัญเมื่อสร้างเนื้อหาเว็บ)
แยกเนื้อหาตามหมวด: สาธารณะ / จำกัดสิทธิ์นักวิจัย / เฉพาะทีมโครงการ
ตามที่ระบุในไฟล์สรุปไอเดีย ห้ามนำข้อมูลดิบ (raw sensor data, โค้ดโมเดล, draft manuscript)
ไปแสดงในส่วนสาธารณะของเว็บ
