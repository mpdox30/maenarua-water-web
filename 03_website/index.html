    การเข้าถึงข้อมูล

    3 ระดับความละเอียดอ่อน

    🟢 สาธารณะ

    กราฟสรุปพยากรณ์รายสัปดาห์ แผนที่ % ความจุอ่าง และข่าวสาร/สรุปงานวิจัย — เปิดให้ทุกคนเข้าถึง

    🟡 จำกัดสิทธิ์ (นักวิจัย/หน่วยงาน)

    Raw sensor data, ค่า NSE/KGE ละเอียด, train/test set, โค้ด/โมเดล และข้อมูล ET/water balance
